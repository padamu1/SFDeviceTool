## 🌐 SFDeviceTool

Device tool for unity.

---

## 📌 How to Import

Window - Pacakge Manager - Install Package from git URL - "https://github.com/padamu1/SFDeviceTool.git"

---

## 📌 How to Use

### Haptic

```csharp
HapticHandler.Haptic(DeviceTool.Haptic.HAPTIC_TYPE.Success);
```
